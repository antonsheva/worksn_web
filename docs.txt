Создание фреймов с обработчиками jqury:

    1 - в "frames" создать класс
    2 - при необходимости, в конструктор передать $id фрейма
    3 - присвоить id внешнему <div>
    3 - названия селекторов обозначить в классах дочерних элементов
    4 - добавить названия селекторов в массив $arr[]
    5 - вызвать -  new jsFuncClick($id, $arr, 'func_name');
    6 - в jsController создать функции обработчики формата:

        funk_name_+название селектора:

               func_name      = "testFunc"
               class элемента = "testClass"
               тогда функция обработчик =  testFunc_testClass()
